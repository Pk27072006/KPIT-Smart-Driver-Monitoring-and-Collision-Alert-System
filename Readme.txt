This repository consists all the information that was used in the project Smart driver monitoring and collision alert system project. The project as completed at KPIT APEX Lab at COEP Pune.
Note: All the final submission of the project is in the main branch of the repository.
Code: The code for the STM32F446RE  is the file named kpit.c. The code was uploaded to STM32 using STM32cubeide.
Report: The containing all the information about hardware and output is in the word doc named Report.docx
PPT: The ppt named FInalPPT.pptx contains the presentation of the project.
